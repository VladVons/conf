Menu \ Центр управления \ Принтеры (Printing) \ Выбрать принтер \ Свойства \ Параметры задания \ Другие параметры (дополнительные)
\ В окошко ввода вставить usb-no-reattach-default \ Добавить
\ В окошко "значение параметра" вставить true \ Применить
